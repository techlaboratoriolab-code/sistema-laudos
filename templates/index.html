Como usar o sistema
1

Obtenha o código da requisição

Localize o código da requisição médica no Aplis do Laboratório ou documento físico.

2

Digite o código no campo abaixo

Insira o código exatamente como aparece na requisição, sem espaços extras.

3

Clique em "Processar Laudo"

O sistema irá buscar o laudo na API e enviar automaticamente via WhatsApp.

4

Acompanhe o resultado

Verifique o status do envio nos logs que aparecerão abaixo do formulário.

Informações importantes:

  • Tempo de processamento: O envio pode levar alguns segundos
  • Formato do código: Aceita códigos alfanuméricos
  • Destinatários: Enviado automaticamente para a lista de contatos configurada
  • Formato do arquivo: Laudos são enviados em PDF

Exemplo de código:

0202042034005

Processando requisição... Aguarde!
